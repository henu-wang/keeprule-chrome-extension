# KeepRule Chrome Extension

A Chrome extension that displays investment wisdom from legendary investors like Warren Buffett, Charlie Munger, and Benjamin Graham on every new tab.

## Features

- Daily investment principles on new tab
- Quotes from 20+ legendary investors
- Clean, minimal design
- Powered by [KeepRule](https://keeprule.com)

## Install

1. Clone this repo
2. Go to `chrome://extensions/`
3. Enable Developer Mode
4. Click "Load unpacked" and select this folder
